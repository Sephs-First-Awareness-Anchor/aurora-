## I. The Five Primitives

These are the necessary and sufficient conditions for any code to function
as a carrier of meaning. They are not cultural, not learned, not chosen.
They are the forced output of single-constraint pressures operating on
any signal system whatsoever.

```
Primitive 1  [X]  →  Presence/Absence
                     The zero/one of all possible codes.
                     Signal present vs signal absent.
                     The ground state of all encoding.
                     Before a language has words it has this.

Primitive 2  [T]  →  Sequence
                     Signals unfold in time.
                     Position in stream is potentially meaningful.
                     The substrate on which all sequential structure is built.
                     Before a language has grammar it has this.

Primitive 3  [N]  →  Differential Cost
                     Not all signals cost the same.
                     Cost gradient shapes which signals survive in a code.
                     The energetic substrate beneath all phonological economy.
                     Before a language has phonology it has this.

Primitive 4  [B]  →  Distinction
                     Signals must be discretely distinguishable from each other.
                     The boundary between signals is as important as the signals.
                     The logical origin of the phoneme as a unit.
                     Before a language has phonemes it has this.

Primitive 5  [A]  →  Selection
                     Signals are chosen, not emitted randomly.
                     Selection is driven by sender state.
                     What separates a signal from a noise.
                     Before a language has meaning it has this.
```

## II. The Collapse Test

Remove any single primitive and the code collapses entirely.

```
Remove [X] → no presence/absence → nothing to transmit
Remove [T] → no sequence → no ordering → no structure possible
Remove [N] → no cost gradient → no selection pressure →
             inventory bloats until distinctions collapse under
             their own maintenance cost
Remove [B] → no discrete signals → receiver cannot distinguish
             anything → all signals collapse into noise
Remove [A] → no selection → signals are noise →
             no information transmitted regardless of signal complexity
```

All five are necessary. Together they are sufficient.
This is the floor beneath the floor.

## III. The Developmental Rule

The five primitives appear in gene string order.
No primitive can function without all prior primitives already active.

```
Presence/Absence  [X]         must exist before Sequence can operate
Sequence          [X·T]       must exist before Differential Cost has a landscape
Differential Cost [X·T·N]     must exist before Distinction is stable
Distinction       [X·T·N·B]   must exist before Selection has options to choose between
Selection         [X·T·N·B·A] completes the minimum viable code
```

A system with only Presence/Absence and Sequence but no Differential Cost
will generate signals but cannot sustain a stable inventory.
A system with Distinction but no Selection produces patterned noise, not communication.
The order is not arbitrary. It is forced.

## IV. Tier II Inheritance — How Pair Pressures Build on Primitives

Every Tier II pressure inherits from exactly the two Tier I primitives
it combines. The derivation is not metaphorical — each pair-pressure
takes the specific demands of its parent primitives and produces the
tension that arises when both must be satisfied simultaneously.

```
[X·T]   Presence/Absence + Sequence
        A signal that was present and is now absent is different from
        one that was never present.
        → Produces: the concept of a completed signal,
          the distinction between present-state and past-state
        → Origin of: PAST REFERENCE — the most primitive
          form of temporal displacement in any code

[X·N]   Presence/Absence + Differential Cost
        The signal that is present must be worth its cost.
        Low-value presence will be selected against.
        → Produces: signal value threshold —
          signals below minimum information value
          are eliminated from the inventory
        → Origin of: REDUNDANCY REDUCTION — the pressure
          that removes uninformative elements from codes over time

[X·B]   Presence/Absence + Distinction
        Presence and absence are themselves a distinction.
        But [X·B] forces a further step — the ability to signal
        the absence of a specific distinction, not just general absence.
        → Produces: targeted negation
        → Origin of: NEGATION — the most primitive
          logical operator in any language

[X·A]   Presence/Absence + Selection
        The sender chooses whether to signal at all.
        Silence is itself a selection.
        → Produces: communicative intentionality —
          the choice to transmit vs withhold
        → Origin of: IMPLICATURE — what is communicated
          by the choice not to signal when signaling was expected

[T·N]   Sequence + Differential Cost
        Position in sequence has a cost profile.
        Earlier positions cost differently than later ones.
        → Produces: positional economy —
          high-information content gets the cheapest positional slot
        → Origin of: INFORMATION STRUCTURE —
          given before new, topic before comment

[T·B]   Sequence + Distinction
        Distinctions must be maintained across sequential positions.
        The same signal must mean the same thing in position 1 and position 7.
        → Produces: cross-positional stability —
          signal identity is preserved across temporal distance
        → Origin of: PROPER NOUNS and REFERENCE TRACKING —
          stable signals for entities that persist across time

[T·A]   Sequence + Selection
        The sender selects different signals based on
        the temporal position of the event being encoded.
        → Produces: temporally-indexed selection
        → Origin of: TENSE — grammatical encoding of event
          position relative to the moment of utterance

[N·B]   Differential Cost + Distinction
        Maintaining a distinction costs energy.
        Only distinctions worth their maintenance cost survive.
        → Produces: the functional load filter —
          distinctions that carry high communicative load
          are maintained, low-load distinctions collapse
        → Origin of: PHONOLOGICAL ECONOMY —
          the sound system retains only distinctions
          whose survival cost is justified by their information value

[N·A]   Differential Cost + Selection
        Selecting a more expensive signal requires
        proportionally more information value to justify it.
        → Produces: cost-proportional selection —
          cheap signals for frequent content,
          expensive signals for rare/high-value content
        → Origin of: ELLIPSIS and PROSODY —
          drop what can be inferred, stress only what is
          informationally necessary

[B·A]   Distinction + Selection
        The sender selects among signals based on
        which distinction needs to be marked.
        Most critically: the distinction between
        self and other is the first distinction
        selection must encode.
        → Produces: boundary-relative selection
        → Origin of: PRONOUNS — grammatical encoding of
          which boundary-position is the relevant reference point
          Origin of: THEORY OF MIND as prerequisite —
          selecting a signal that marks self/other requires
          modeling the other as a distinct bounded agent
```
