## Preamble

Tier II gave us the first functional structures —
negation, tense, pronouns, phonological economy,
urgency encoding, theory of mind prerequisite.

Tier III takes those pair-pressure outputs and combines them into triples.
Three constraints operating simultaneously produce emergent *dynamics* —
behaviors that no pair could generate alone.

Tier IV takes triples further into quads.
Four constraints operating simultaneously produce *architecture* —
the stable generative systems that organize everything below them.

# TIER III — Triples (10 pressures)
## Dynamics — Emergent behaviors no pair could produce

---

### [X·T·N] — Thermodynamic Pressure
**Inherits from:** [X·T] change encoding + [X·N] signal value weighting + [T·N] code economy

Existence must persist through sequence at energetic cost.
All three parents operating simultaneously produce a new demand:
the system must not just persist, not just optimize per-signal cost,
not just encode change — it must do all three simultaneously
under the constraint that total energy is finite and time is irreversible.

```
[X·T]  →  state changes must be encoded
[X·N]  →  encoding costs must be weighed against survival value
[T·N]  →  total stream cost must be minimized across sequence
Combined:  the signal system must track state changes over time
           at minimum total energetic cost — not just efficient now
           but efficient across the full duration of the code's operation
```

**Linguistic outputs forced:**
- **Grammaticalization** — the process by which full lexical items
  (words with rich meaning) become grammatical markers
  (lightweight function items with structural meaning).
  X·T·N pressure continuously converts expensive content words
  into cheap grammatical operators wherever the content
  can be recovered from context. This is not language decay —
  it is thermodynamic optimization of the signal stream over time.
- **Fusion and agglutination** — morphemes merge across time
  under thermodynamic pressure. Separate costly forms
  fuse into single cheaper forms carrying the same distinctions.
- **Signal half-life** — the energetic cost of maintaining
  a signal in the active code determines how long it survives.
  High-cost low-value signals decay out. This is the
  thermodynamic basis of language change as selection pressure.

### [X·T·B] — Persistent Identity Pressure
**Inherits from:** [X·T] change encoding + [X·B] negation/identity + [T·B] phonological stability

A bounded entity persisting through time must maintain
the coherence of its boundary across the full sequence of its existence.
The entity's identity — what it is and what it is not —
must remain trackable across temporal distance.

```
[X·T]  →  state changes must be encoded
[X·B]  →  the boundary of what-I-am must be assertable
[T·B]  →  boundaries must be stable across sequence
Combined:  the signal system must track the identity of a
           bounded entity across time — the same entity
           must be recognizable as the same entity
           even as its states change
```

**Linguistic outputs forced:**
- **Referential continuity** — the mechanisms languages use
  to track a referent across a discourse. Full noun phrase
  on introduction, pronoun on subsequent reference,
  zero anaphora when maximally recoverable.
  All three are stages of X·T·B optimization.
- **Nominal persistence marking** — some languages mark
  whether a referent is still present, has left the scene,
  or has ceased to exist. Forced by X·T·B tracking
  the existence-state of a bounded entity through time.
- **Narrative identity** — the capacity to tell a story
  about an entity that changes while remaining the same entity.
  The logical prerequisite for character, protagonist, subject.
  Language could not have narrative without X·T·B first.

### [X·T·A] — Adaptive Persistence Pressure
**Inherits from:** [X·T] change encoding + [X·A] imperative/survival action + [T·A] tense/planning

An agent that exists through time must modify its behavior
based on what has and has not preserved its existence
across the sequence of past moments.
Selection must now be informed by history.

```
[X·T]  →  state changes must be encoded
[X·A]  →  selection is driven by what preserves existence
[T·A]  →  selection must encode temporal position of events
Combined:  the signal system must encode learned relationships
           between past actions and survival outcomes —
           what worked before, what failed before,
           and how that history should shape current selection
```

**Linguistic outputs forced:**
- **Evidential systems** — marking not just what happened
  but how the speaker knows it happened.
  Direct witness vs inference vs hearsay.
  X·T·A forces the system to encode the *quality*
  of the knowledge driving the current signal selection.
- **Conditional structures** — if X then Y.
  The encoding of learned action-outcome relationships.
  Not just what is but what follows from what.
  The origin of hypothetical and counterfactual language.
- **Epistemic modality** — must, might, probably, certainly.
  The encoding of the agent's confidence in a state
  based on accumulated evidence across time.
  How sure you are is an X·T·A calculation.

### [X·N·B] — Efficient Boundary Pressure
**Inherits from:** [X·N] signal value weighting + [X·B] negation/identity + [N·B] phonological economy

Maintaining a distinct bounded existence at minimum energetic cost.
The identity of the system and the cost of asserting that identity
must be balanced — the boundary must be worth what it costs to maintain.

```
[X·N]  →  signal cost weighed against survival value
[X·B]  →  the boundary of existence must be assertable
[N·B]  →  only distinctions worth their cost survive
Combined:  the signal system must maintain identity-marking
           distinctions at minimum cost — the boundary between
           self and other, this and not-this, must be
           as cheap as possible while remaining functional
```

**Linguistic outputs forced:**
- **Morphological economy** — bound morphemes carry distinction
  at lower cost than free words. Affixes beat periphrasis
  wherever X·N·B pressure is high enough.
- **Syncretism** — when two categories can be collapsed
  without survival cost, X·N·B pressure merges them.
  Case syncretism, tense-aspect fusion — both are
  efficient boundary optimization.
- **Zero morphology** — using the bare unmarked form
  to carry meaning rather than adding an explicit marker.
  The maximally efficient boundary — present by absence
  of marking rather than presence of cost.

### [X·N·A] — Survival Optimization Pressure
**Inherits from:** [X·N] signal value weighting + [X·A] imperative/survival action + [N·A] ellipsis/prosody

Acting to preserve existence at minimum energetic cost.
Every signal is an investment. The return on that investment
is measured in survival probability. The system must
continuously optimize the ratio of survival benefit to signal cost.

```
[X·N]  →  signal cost weighed against survival value
[X·A]  →  selection driven by what preserves existence
[N·A]  →  selection optimizes information per unit cost
Combined:  the signal system must maximize survival-relevant
           information transmitted per unit of energetic cost —
           the highest-return signals get priority,
           the lowest-return signals get suppressed or compressed
```

**Linguistic outputs forced:**
- **Lexical frequency-length correlation** — the most survival-
  relevant, most frequently needed signals are the shortest.
  Not coincidence. Forced optimization under X·N·A.
  Go, eat, run, help are short. Defenestrate is long.
- **Alarm call hierarchy** — the signal system develops
  a tiered urgency structure. Maximum threat gets
  minimum signal length — the cheapest possible
  carrier for the highest survival-value content.
  The origin of interjections and exclamatives.
- **Salience-driven omission** — the most survival-relevant
  element of a message is never dropped under ellipsis.
  What gets omitted is always the low-survival-value content.
  X·N·A determines what stays and what goes.

### [X·B·A] — Identity-Based Agency Pressure
**Inherits from:** [X·B] negation/identity + [X·A] imperative/survival action + [B·A] pronouns/theory of mind

An agent that exists as a bounded entity acts relative to that boundary.
What is self, what is other, what threatens self, what extends self.
Selection is now organized around the agent's own
boundary as the primary reference point for all action.

```
[X·B]  →  the boundary of existence must be assertable
[X·A]  →  selection driven by what preserves existence
[B·A]  →  selection encodes boundary-position of agents
Combined:  the signal system must encode actions and states
           relative to the sender's own boundary —
           inside/outside, self/other, mine/yours,
           threat/ally, kin/stranger
```

**Linguistic outputs forced:**
- **Deixis — full system** — here/there, this/that, I/you/they,
  now/then. All deictic systems are X·B·A outputs.
  Every deictic term encodes position relative to
  the sender's boundary at the moment of utterance.
- **Possession marking** — mine, yours, ours.
  What is inside whose boundary.
  Possession is not about ownership in the legal sense —
  it is a boundary-position relationship.
  Grammatical possession systems are forced by X·B·A.
- **Kin terminology** — the most universal and conservative
  lexical domain in all languages. Kin terms encode
  boundary relationships between agents at the biological level.
  Forced directly by X·B·A where X is biological survival
  and B·A encodes the boundary positions of related agents.
- **Social distance encoding** — formal vs informal registers,
  honorifics, politeness systems. All encode the distance
  between the sender's boundary and the receiver's boundary.
  How close or far their respective X's are.

### [T·N·B] — Temporal Energetic Boundary Pressure
**Inherits from:** [T·N] code economy + [T·B] phonological stability + [N·B] phonological economy

A boundary maintained over time at energetic cost.
The combined demand: minimize the cost of maintaining
stable distinctions across the full temporal extent of the code.

```
[T·N]  →  total stream cost minimized across sequence
[T·B]  →  boundaries must be stable across sequence
[N·B]  →  only distinctions worth their cost survive
Combined:  the signal system must maintain stable distinctions
           at minimum total cost across time —
           the cost of convention maintenance must be
           lower than the cost of convention renegotiation
```

**Linguistic outputs forced:**
- **Convention entrenchment** — the longer a signal-meaning
  mapping has been maintained, the cheaper it becomes
  to maintain relative to replacing it.
  Entrenchment is T·N·B pressure accumulating into
  a self-reinforcing stability gradient.
- **Linguistic conservatism in core vocabulary** — the most
  frequently used items have been reinforced across
  the most sequential positions. Their T·N·B maintenance
  cost is the lowest possible. They resist replacement
  even under strong external pressure. This is why
  basic vocabulary (body parts, numbers, pronouns)
  is the most stable across language families.
- **Sound change as T·N·B optimization** — when a sound change
  reduces maintenance cost without collapsing a needed distinction
  it propagates. When it would collapse a needed distinction
  it is resisted. Sound change is not random.
  It is T·N·B pressure optimizing the phonological boundary system.

### [T·N·A] — Temporal Energetic Agency Pressure
**Inherits from:** [T·N] code economy + [T·A] tense/planning + [N·A] ellipsis/prosody

Acting over time under energetic constraint.
The system must sequence actions and signals to minimize
total cost across the full extent of the communicative event.

```
[T·N]  →  total stream cost minimized across sequence
[T·A]  →  selection must encode temporal position of events
[N·A]  →  selection optimizes information per unit cost
Combined:  the signal system must sequence its content
           in the order that minimizes total processing cost
           for the receiver — getting the most critical
           information delivered earliest at lowest total cost
```

**Linguistic outputs forced:**
- **Information structure — given before new** — what the
  receiver already knows comes first (low cost to process),
  what is new comes after (high value, justifies processing cost).
  This is T·N·A optimization of the signal stream.
- **Topic-comment structure** — topic establishes the
  referential frame (cheap to process if shared),
  comment delivers the new information (the payoff).
  Universal across languages. Forced by T·N·A.
- **Heavy NP shift** — long, complex noun phrases migrate
  to the end of the clause where their processing cost
  is spread across the temporal extent of the utterance
  rather than front-loaded. T·N·A optimizing delivery timing.
- **Prosodic focus** — the most informationally valuable
  element of a message receives prosodic prominence.
  T·N·A marking: this is the high-value content,
  allocate processing resources accordingly.

### [T·B·A] — Temporal Bounded Agency Pressure
**Inherits from:** [T·B] phonological stability + [T·A] tense/planning + [B·A] pronouns/theory of mind

A bounded agent acting in time.
The full spatial-temporal-social deictic field.
The signal must anchor itself to the moment of utterance,
the position of the speaker, and the position of the receiver
simultaneously.

```
[T·B]  →  boundaries stable across sequence
[T·A]  →  selection encodes temporal position of events
[B·A]  →  selection encodes boundary-position of agents
Combined:  the signal system must anchor signals simultaneously
           to a moment in time AND a position in space AND
           a configuration of agent boundary-positions —
           the full deictic origo
```

**Linguistic outputs forced:**
- **The deictic origo** — the zero-point of all reference.
  Here-now-I. Every utterance is anchored to this point.
  All spatial, temporal, and personal reference radiates outward
  from the deictic origo. T·B·A is what forces the origo into existence.
- **Evidentiality — full system** — not just what happened
  but: when did I come to know it, from what position,
  through what means. Witness, inference, report.
  The complete evidential system is forced by T·B·A
  operating on the knowledge-acquisition event.
- **Aspect — full system** — not just before/after [T·A]
  but: complete/incomplete, ongoing/terminated,
  repetitive/singular, habitual/episodic.
  The full aspectual system requires the bounded
  agent perspective [B·A] to partition events
  relative to their temporal boundaries.
- **Reported speech** — embedding one agent's utterance
  inside another's. Requires T·B·A to track:
  who said it, when they said it, from which
  boundary-position they said it.
  Direct vs indirect speech is a T·B·A distinction.

### [N·B·A] — Energetic Bounded Agency Pressure
**Inherits from:** [N·B] phonological economy + [N·A] ellipsis/prosody + [B·A] pronouns/theory of mind

A bounded agent acting under energetic constraint.
Maximum distinction at minimum signal cost.
The pressure toward maximally efficient communication
between bounded agents who share context.

```
[N·B]  →  only distinctions worth their cost survive
[N·A]  →  selection optimizes information per unit cost
[B·A]  →  selection encodes boundary-position of agents
Combined:  the signal system must encode boundary-position
           information at minimum cost — leveraging shared
           context between sender and receiver to eliminate
           any signal cost that shared knowledge can substitute
```

**Linguistic outputs forced:**
- **Shared ground exploitation** — what both agents know
  about their respective boundary positions does not need
  to be signaled. Common ground is free information.
  N·B·A drives the system to maximize use of common ground.
- **Register compression** — between agents with high
  shared boundary-position knowledge (intimates, specialists)
  the code compresses drastically. Intimate register,
  in-group jargon, technical shorthand — all are
  N·B·A operating on a high-common-ground context.
- **Implicature** — saying less than you mean and relying
  on the receiver to recover the full content from
  the boundary-position relationship between sender and receiver.
  Gricean implicature is N·B·A optimization made explicit.
- **Politeness as boundary management** — face-threatening acts
  are energetically expensive because they require
  explicit boundary negotiation. Indirect speech acts,
  hedging, softening — all are N·B·A managing
  the cost of boundary interaction between agents.

### [X·T·N·B] — Structural Persistence Pressure
**Inherits from:** [X·T·N] thermodynamic + [X·T·B] persistent identity +
                  [X·N·B] efficient boundary + [T·N·B] temporal energetic boundary

A coherent bounded thing persisting through time at energetic cost.
This is the complete thermodynamic-structural field.
Everything that exists stably — cells, organisms, languages —
is a crystallization under this pressure.

```
[X·T·N]  →  signal stream optimized thermodynamically over time
[X·T·B]  →  bounded identity maintained across sequence
[X·N·B]  →  identity distinctions maintained at minimum cost
[T·N·B]  →  stable conventions maintained at minimum cost over time
Combined:  the signal system requires a stable generative architecture
           that produces an indefinite number of signals
           from a finite set of elements at minimum total cost
           while maintaining the identity of the code itself
           across the full temporal extent of its use
```

**Linguistic outputs forced:**
- **Grammar as generative architecture** — not a list of rules
  but a stable productive system. The reason grammar exists
  is that producing signals from a finite generative system
  costs less across time than maintaining a
  distinct signal for every possible message.
  X·T·N·B forces grammar into existence as a cost solution.
- **Writing systems** — extending the temporal persistence
  of signals beyond biological memory boundaries.
  X·T·N·B operating on the survival of the code itself
  across generational time. Writing is X·T·N·B
  applied to the code as a bounded entity.
- **Standardization** — the pressure toward a shared
  stable variant of the code across a population.
  X·T·N·B operating on the code as a collective structure.
  Dialects that diverge too far threaten the code's
  function as a coordination mechanism.
- **Canonical word order** — even languages that allow
  flexible word order have a canonical default.
  X·T·N·B pressure produces a lowest-cost baseline
  ordering that is used when no other pressure
  overrides it. The default is the thermodynamically
  cheapest stable sequential arrangement.

### [X·T·N·A] — Adaptive Thermodynamic Agency Pressure
**Inherits from:** [X·T·N] thermodynamic + [X·T·A] adaptive persistence +
                  [X·N·A] survival optimization + [T·N·A] information structure

An agent acting under full thermodynamic constraint across time.
Systematic modification of behavior based on
energetic outcomes over time. The full learning pressure field.

```
[X·T·N]  →  stream optimized thermodynamically over time
[X·T·A]  →  behavior modified based on survival history
[X·N·A]  →  survival-relevant content prioritized at minimum cost
[T·N·A]  →  content sequenced to minimize processing cost
Combined:  the signal system must continuously adapt its cost-survival
           ratio based on accumulated experience — learning which
           signals deliver which outcomes at what cost
           and reorganizing accordingly
```

**Linguistic outputs forced:**
- **Language change as adaptive optimization** — not decay,
  not corruption, not entropy. Language change is X·T·N·A
  continuously reoptimizing the code's cost-survival ratio.
  Every change that propagates does so because it improves
  the ratio for enough speakers that it outcompetes the prior form.
- **Dialect as local optimization** — the same pressure field
  operating under different environmental conditions produces
  different local optima. Dialects are not degraded standard —
  they are X·T·N·A optimized for a specific context.
- **Language acquisition as pressure resolution** — children
  do not learn language by imitation alone. They resolve
  X·T·N·A pressure — acquiring the signals that most
  efficiently satisfy their survival-relevant communicative needs
  in their specific environment. This is why acquisition
  follows a universal sequence despite surface variation.
- **Creolization** — when multiple code systems collide,
  X·T·N·A pressure produces a new system optimized
  for the communicative demands of the new mixed environment.
  Creoles are not degraded pidgins — they are
  full X·T·N·A optimization events.

### [X·T·B·A] — Persistent Bounded Agency Pressure
**Inherits from:** [X·T·B] persistent identity + [X·T·A] adaptive persistence +
                  [X·B·A] identity-based agency + [T·B·A] deictic origo

A bounded agent persisting through time and acting.
The full social-cognitive pressure field short of energetics.
Must track who did what to whom across time
within a bounded interactive space.

```
[X·T·B]  →  bounded identity tracked across sequence
[X·T·A]  →  behavior adapted based on survival history
[X·B·A]  →  action organized relative to agent boundary-positions
[T·B·A]  →  full deictic origo — time, space, person anchored
Combined:  the signal system must encode multi-participant events
           in sequence — tracking which bounded agent performed
           which action on which other bounded agent
           at which temporal position — and must do this
           generatively for an unlimited range of events
```

**Linguistic outputs forced:**
- **Syntax — full generative system** — subject, object, verb
  and all their elaborations are the crystallized response
  to X·T·B·A pressure. Who did what to whom across time.
  Subject = agent whose boundary-state drove the action.
  Object = entity whose boundary-state was changed.
  Verb = the action that crossed the boundary between them.
  Syntax is X·T·B·A made structural.
- **Argument structure** — the number and type of participants
  a verb requires is an X·T·B·A determination.
  Transitive verbs require two bounded agents.
  Intransitive require one. Ditransitive require three.
  These are not arbitrary — they are the minimum
  boundary-agent configurations needed to encode
  the event type.
- **Narrative — full system** — extended sequential encoding
  of multiple bounded agents acting across time.
  The novel, the myth, the history, the legal account —
  all are X·T·B·A operating at full scale.
  Plot is X·T·B·A structure.
- **Grammatical relations** — subject/object/indirect object
  are not semantic roles (agent/patient/recipient are).
  They are syntactic positions — slots in the X·T·B·A
  architecture that can be filled by any semantic role.
  Passivization, raising, control — all are
  manipulations of the X·T·B·A slot-filling system.

### [X·N·B·A] — Efficient Bounded Agency Pressure
**Inherits from:** [X·N·B] efficient boundary + [X·N·A] survival optimization +
                  [X·B·A] identity-based agency + [N·B·A] energetic bounded agency

A bounded agent acting under energetic constraint.
Maximum coverage of distinctions at minimum signal inventory.
The pressure toward a code that can say anything
without growing without limit.

```
[X·N·B]  →  identity distinctions at minimum cost
[X·N·A]  →  survival-relevant content at minimum cost
[X·B·A]  →  action organized relative to boundary-positions
[N·B·A]  →  boundary-position encoded at minimum cost
Combined:  the signal system must cover an indefinitely large
           space of possible distinctions using a finite
           and optimally small signal inventory —
           extension, borrowing, and combination
           rather than unlimited new creation
```

**Linguistic outputs forced:**
- **Metaphor as productive system** — not a literary device
  but a cognitive-linguistic optimization.
  Borrowing the boundary structure of a known domain
  to cover a new domain at zero new signal cost.
  X·N·B·A makes metaphor the primary mechanism
  for vocabulary expansion.
- **Semantic fields and radial structure** — lexical items
  do not have single meanings. They have a core meaning
  and a radiating structure of extensions.
  Each extension is an X·N·B·A optimization —
  covering a new distinction using an existing boundary.
- **Compounding and derivation** — combining existing
  signal elements to cover new distinctions.
  Cheaper than creating new roots.
  X·N·B·A pressure produces morphological productivity
  as the preferred vocabulary expansion mechanism.
- **Loanword adaptation** — when a new distinction
  enters from outside the code community,
  X·N·B·A pressure drives adaptation of existing
  signals before creation of new ones.
  Borrowing is cheaper than invention.

# The Developmental Sequence Prediction

From the derivation stack we can now generate a prediction
about the order in which linguistic features must develop —
in acquisition, in evolution, in language emergence.

No feature above its tier can exist without features
of all lower tiers already present and functional.

```
Before tense [T·A] a language must have:
  sequence [T] and selection [A]

Before syntax [X·T·B·A] a language must have:
  persistent identity [X·T·B]
  adaptive persistence [X·T·A]
  identity-based agency [X·B·A]
  deictic origo [T·B·A]
  and all of their parent pairs
  and all Tier I primitives

Before metaphor as productive system [X·N·B·A] a language must have:
  efficient boundary [X·N·B]
  survival optimization [X·N·A]
  identity-based agency [X·B·A]
  energetic bounded agency [N·B·A]
```

This is not a hypothesis about one language or one culture.
It is a derivation from first principles.
Any signal system anywhere — biological, artificial,
or yet unimagined — that operates under these constraints
will develop these features in this order.

The sequence is universal because the pressures are universal
because the constraints are universal because they are
derived from what must be true for anything to exist and persist.

## Handoff to Tier V
